Installing on Mac OS X

  1. If you already have everything needed for pre-install, you can skip steps up to this step.
  2. Install MacPorts (versions for Leopard, Tiger).
  3. Now enter Terminal (/Applications/Utilities/Terminal) and enter commands to install Python: 
    sudo port install wget python27 python-select py27-pip
    Select Python 2.7 as main python interpreter: 
    sudo python_select python27
  4. Install MySQL server and set up it if you don't have any SQL database: 
    sudo port install mysql5-server && sudo -u _mysql mysql_install_db5
    This creates new MySQL user 'root' with no password: 
    sudo /opt/local/lib/mysql5/bin/mysqladmin -u root password ''
  5. Install Memcached and python binding for it 
    sudo port install memcached
  6. Download chieftain and unpack it 
    cd ~/Downloads && wget https://github.com/paulmillr/chieftain/tarball/master
  7. Install all pythonic chieftain dependencies: 
    sudo pip-2.7 install -r pip-req.txt
  8. Enter chieftain directory. Copy local_settings.example.py to local_settings.py and edit it for your configuration: 
    mv local_settings.example.py local_settings.py && mate local_settings.py
    Sync database data: 
    python manage.py syncdb
  9. Now run server: 
    python manage.py runserver 127.0.0.1:8000
  10. Open browser and go to 127.0.0.1:8000. It works!
